
# GUN-TACTYX

The subject is GUN-TACTYX which is found here: https://apocalyx.sourceforge.net/guntactyx/downloads.php
This is a playground, do not expect anything to be working.

GUN-TACTYX is written in the lanugate Borland C++, primarely for windows.
It seems the source code is missing bits, and that the newer APOCALYPS game engine has the bits, but is newer and not compatible.
Too much trouble to port the game as is.
It is a pity; I loved the game and would have liked to see a modern version.
