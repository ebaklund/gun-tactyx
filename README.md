Project Overview: GUN-TACTYX Re-envisioned

This toy project is a rewrite of the original GUN-TACTYX,
which can be found here. https://apocalyx.sourceforge.net/guntactyx/downloads.php

Please note: this is a personal playground; do not expect a fully functional build.

The original GUN-TACTYX was written in Borland C++ specifically for Windows.
Attempting to port the game in its current state proved to be a dead end for a few reasons:

- Missing Source Code: The original source appears to be incomplete.
- Incompatibility: While the newer APOCALYPS engine seems to contain the missing components,
  it has evolved significantly and is no longer backward compatible with the original game logic.

Ultimately, porting the legacy code "as-is" was more trouble than it was worth.
It’s a pity, as I loved the original game and always wanted to see a modern version.

This playground is an just a result of my personal learning what it takes to make such a masterpiece as GUN-TACTYX.
